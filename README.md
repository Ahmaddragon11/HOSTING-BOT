# 🤖 BotForge v4.0 — Personal Bot Hosting Manager
### by AHMADDRAGON

مدير البوتات الشخصي — استضف، راقب، وتحكم في بوتاتك بسهولة تامة من Telegram.

---

## 🚀 التشغيل

```bash
python main.py
```

لا تحتاج لأي تثبيت يدوي — كل شيء يُثبَّت تلقائياً عند أول تشغيل.

---

## 📁 هيكل المشروع

```
botforge_v4/
├── main.py                  ← نقطة التشغيل الرئيسية
├── requirements.txt
│
├── core/                    ← المنطق الأساسي
│   ├── config.py            ← الإعدادات والثوابت
│   ├── models.py            ← نموذج بيانات BotInstance
│   ├── process_manager.py   ← إدارة دورة حياة البوتات
│   ├── scheduler.py         ← جدولة المهام
│   └── notifier.py          ← نظام الإشعارات
│
├── handlers/                ← معالجات Telegram
│   ├── router.py            ← تسجيل جميع الـ handlers
│   ├── base.py              ← كلاس أساسي مشترك
│   ├── panel.py             ← لوحة التحكم الرئيسية
│   ├── bot_mgr.py           ← إضافة/تشغيل/إيقاف/تحديث البوتات
│   ├── bot_ctrl.py          ← التحكم في هوية البوتات
│   ├── env_mgr.py           ← إدارة متغيرات البيئة
│   ├── scheduler_h.py       ← واجهة الجدولة
│   ├── search_h.py          ← البحث والتصفية
│   ├── notif_h.py           ← إعدادات الإشعارات
│   └── media_h.py           ← معالجة الصور والملفات
│
├── utils/                   ← أدوات مساعدة
│   ├── logger.py            ← إعداد السجلات
│   ├── extractor.py         ← استخراج الملفات المضغوطة
│   ├── bot_controller.py    ← التحكم في بوتات Telegram
│   └── keyboards.py         ← مولّد لوحات المفاتيح
│
├── data/                    ← بيانات البوت (تُنشأ تلقائياً)
│   ├── state.json           ← حالة البوتات
│   ├── schedules.json       ← المهام المجدولة
│   ├── notifications.json   ← إعدادات الإشعارات
│   └── env_vars.json        ← متغيرات البيئة
│
├── hosted_bots/             ← مجلدات البوتات المستضافة
└── botforge_logs/           ← سجلات النظام والبوتات
```

## ✨ المميزات

| الميزة | الوصف |
|--------|-------|
| 📤 رفع الملفات | `.zip` / `.tar.gz` / `.py` — مباشرة في الشات |
| 🐍 بيئة افتراضية | `venv` خاصة لكل بوت تلقائياً |
| 📦 اكتشاف المتطلبات | من `requirements.txt` أو اكتشاف تلقائي للـ imports |
| ▶️ تشغيل / إيقاف | أزرار inline أو أوامر نصية |
| 🔄 إعادة تشغيل تلقائي | عند الانهيار مع عداد |
| 📊 إحصائيات | CPU، RAM، Uptime، PID لكل بوت |
| 📋 سجلات | عرض، بحث، مسح، تحديث فوري |
| 🔄 تحديث الكود | رفع ملف جديد بدون حذف البوت |
| 🔑 متغيرات البيئة | إدارة كاملة لكل بوت من داخل Telegram |
| 📅 الجدولة | تشغيل/إيقاف في أوقات محددة أو بشكل دوري |
| 🔔 إشعارات | تنبيهات مخصصة عند كل حدث |
| 🔍 بحث وتصفية | بحث بالاسم، ID، الحالة |
| 🛠 إعدادات البوتات | تغيير اسم/وصف/نبذة/صورة أي بوت |
| ⚙️ إعدادات BotForge | التحكم في هوية البوت الرئيسي |
| ↗️ زر الانتقال السريع | فتح البوت المستضاف مباشرة |
| 💾 حالة دائمة | تُحفظ وتُحمَّل عند كل إعادة تشغيل |
| ⛔ أمان | خاص بالمالك فقط (Owner ID) |

## 📅 صيغ الجدولة

```
08:30              ← يومياً الساعة 8:30 صباحاً
08:30 fri          ← كل جمعة 8:30 صباحاً
every 2h           ← كل ساعتين
every 30m          ← كل 30 دقيقة
once 2024-12-31 23:59  ← مرة واحدة في تاريخ محدد
```

## ⌨️ الأوامر

| الأمر | الوظيفة |
|-------|---------|
| `/start` | الصفحة الرئيسية |
| `/panel` | لوحة التحكم |
| `/bots` | قائمة البوتات |
| `/add` | إضافة بوت جديد |
| `/search [نص]` | بحث في البوتات |
| `/stats` | إحصائيات النظام |
| `/logs <id> [بحث]` | سجلات بوت |
| `/start_bot <id>` | تشغيل بوت |
| `/stop_bot <id>` | إيقاف بوت |
| `/restart <id>` | إعادة تشغيل |
| `/schedule` | جدولة مهمة |
| `/cancel` | إلغاء العملية الحالية |
