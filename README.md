# 🤖 BotForge v4.0 — Personal Bot Hosting Manager
### by AHMADDRAGON

مدير البوتات الشخصي — استضف، راقب، وتحكم في بوتاتك بسهولة تامة من Telegram.

---

## 🚀 التثبيت والتشغيل

### المتطلبات الأساسية:
- Python 3.8+
- حساب Telegram Bot (من @BotFather)

### خطوات التثبيت:

1. **استنساخ المشروع:**
   ```bash
   git clone https://github.com/Ahmaddragon11/HOSTING-BOT.git
   cd HOSTING-BOT
   ```

2. **تثبيت المتطلبات:**
   ```bash
   pip install -r botforge/requirements.txt
   ```

3. **إعداد المتغيرات البيئية:**
   أنشئ ملف `.env` في مجلد `botforge/`:
   ```env
   BOTFORGE_TOKEN=your_bot_token_here
   BOTFORGE_OWNER=your_telegram_user_id
   ```

4. **تشغيل البوت:**
   ```bash
   cd botforge
   python main.py
   ```

## 🌐 النشر

### على خادم (مثل VPS أو Heroku):

1. **رفع الملفات:**
   - ارفع جميع ملفات `botforge/` إلى الخادم
   - لا ترسل مجلد `.venv` أو ملفات `.git`

2. **إعداد البيئة:**
   - قم بتثبيت Python والمتطلبات
   - أضف المتغيرات البيئية أو ملف `.env`

3. **تشغيل مستمر:**
   - استخدم `screen` أو `tmux` للتشغيل في الخلفية
   - أو استخدم systemd للخدمة

### ملاحظات مهمة:
- تأكد من أن التوكن صالح والبوت لديه صلاحيات كافية
- راقب استخدام الموارد (CPU/RAM) خاصة مع بوتات متعددة
- احتفظ بنسخة احتياطية من مجلد `data/` بانتظام
